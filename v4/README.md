# Semperficon v4 Static Website

Modern multi-page website build for Semperficon Construction using pure HTML5, CSS3, and Vanilla JavaScript.

## Files

- `index.html` - Home page with hero, stats, service overview, process, and CTA
- `about.html` - Company profile, values, and timeline
- `services.html` - Detailed service overview and delivery model
- `contact.html` - Contact details and consultation guidance
- `style.css` - Shared global styles, layout system, components, and animations
- `script.js` - Shared behavior (mobile nav, active links, reveal effects, counters, footer year)
- `assets/images/logo-semperficon.svg` - Company logo used on all pages

## Quick Start

Open `index.html` in any browser.

## Notes

- No backend is used.
- Contact interactions are mailto-only.
- All pages share one stylesheet and one script for maintainability.
